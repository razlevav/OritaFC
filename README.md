# ORITA – מערכת תמחור מנות

אפליקציית web מקומית לתמחור מנות, פריטים מוכנים/רטבים ומרכיבי גלם, בעברית מלאה ו-RTL.

## הרצה

```bash
npm install   # פעם ראשונה בלבד
npm run dev
```

זה מריץ שני תהליכים במקביל:
- שרת API מקומי על `http://localhost:4000` (שומר את הנתונים בקובץ `server/data.json`)
- שרת הפיתוח של Vite על `http://localhost:5173` (זה הכתובת שנפתחים בדפדפן)

הנתונים נשמרים בקובץ `server/data.json` ונשארים גם אחרי סגירת הדפדפן/המחשב.

## מבנה

- `server/` – Express API + אחסון JSON מקומי (ingredients, prepared-items, dishes)
- `src/lib/calc.js` – כל לוגיקת חישוב העלויות והיחידות
- `src/components/` – מסכי המערכת (דשבורד, עריכת מנה, מרכיבי גלם, רטבים)
- `branding/` – חומרי המיתוג המקוריים של ORITA (לוגו PDF/PNG)
- `public/branding/` – גרסאות הלוגו המשמשות בפועל בממשק
